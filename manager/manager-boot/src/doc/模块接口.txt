数据源配置
    数据源类型字典：DataSource
        接口:/dict/all(get)
        注释:查询全部数据字典值
    展示页面接口：/datasource(get)
    条件查询接口：/datasource(get)
    查看详细信息接口：
        路径：/datasource/{id}(get)
        目标表：DataSource 、DataSourcePlugin
        注释：DataSource : DataSourcePlugin = 1 : n
    数据保存页面展示字段:
        路径：/dicdatasourceplugin/{sourceType}(get)
        注释：选择完具体的数据源，根据不同的数据源类型，显示不同的页面展示字段（fieldName,fieldCode,fieldOrder,fieldType,fieldTypeKey）
    数据保存接口：
        路径：/datasource(post)
        目标表：DataSource 、DataSourcePlugin
        注释：DataSource : DataSourcePlugin = 1 : n
    数据逻辑删除接口:
        路径：/datasource/{id}(delete)
        字段：iscancel

全局配置
    告警策略字典：AlertPlugin 接口：/dict/all(get)
    展示页面接口：/alarm/info(get)
    邮件方式字典：/dicalarmplugin/{alerttype}(get)
    数据保存接口：/alarm(post)
    用户人员接口：/cuser(get) 分页数据
    需前端处理实体字段：alarmType、alarmPlugins{alarmType、pluginCode、pluginName、pluginValue}、alarmUsers{userId}
    
日志级别：LogLevelEnum
        查询当前日志级别：
            接口：/loggrade(get)
            注释：根据id排序，取最大id数据
        新增/修改日志界别：
            接口：/loggrade(post)
            注释：若loggrade中有id则为修改，没有则为新增。
